    GStreamer RTP和 RTSP

    RTP（传输层）

    RTP全名是Real-time Transport Protocol（实时传输协议）。RTP 用来为IP网上的语音、图像、传真等多种需要实时传输的多媒体数据提供端到端的实时传输服务。RTP为Internet上端到端的实时传输提供时间信息和流同步，但并不保证服务质量，服务质量由 RTCP来提供。

    将数据封装为RTP包

    RTP:Server UDP

    jetson nano：作为服务器端

    IP:10.0.0.179

gst-launch-1.0 videotestsrc is-live=true ! jpegenc ! rtpjpegpay ! udpsink host=10.0.0.215 port=5004

    host是客户端地址

    1. videotestsrc is-live=true：
    2. videotestsrc 是一个视频测试源，它生成一个测试图案（例如彩条屏）。这在测试和调试视频流管道时非常有用。
    3. is-live=true 表示源应该表现得像一个实时源。这意味着 videotestsrc 会尝试按照实时时间生成每一帧，而不是尽可能快地生成。
    4. jpegenc：
    5. jpegenc 是一个 JPEG 编码器，它将 videotestsrc 生成的原始视频帧编码成 JPEG 格式。这是一个压缩步骤，可以减少数据的大小，便于传输。
    6. rtpjpegpay：
    7. rtpjpegpay 是一个 RTP JPEG 负载器。它将 jpegenc 生成的 JPEG 帧封装成 RTP 包。RTP（实时传输协议）是一种用于传输实时数据，如音频和视频的网络协议。
    8. udpsink host=10.0.0.215 port=5004：
    9. udpsink 是一个网络接收器，用于通过网络发送数据。
    10. host=10.0.0.215 指定了目标主机的 IP 地址，即视频流将被发送到这个 IP 地址。
    11. port=5004 指定了用于传输数据的 UDP 端口号。

    RTP:Client UDP

    mac 录屏台式机：客户端

    IP:10.0.0.215

gst-launch-1.0 udpsrc port=5004 ! application/x-rtp,encodingname=JPEG,payload=26 ! rtpjpegdepay ! jpegdec ! autovideosink
    1. udpsrc port=5004：
    2. udpsrc 是一个网络源，用于从网络接收数据。
    3. port=5004 指定了 UDP 端口号，这是 udpsrc 用来监听入站数据的端口。
    4. application/x-rtp,encodingname=JPEG,payload=26：
    5. 这部分定义了 udpsrc 接收数据的具体格式（caps）。application/x-rtp 表示数据采用 RTP（实时传输协议）格式。
    6. encodingname=JPEG 指出这个 RTP 流是使用 JPEG 编码的视频数据。
    7. payload=26 指定了 RTP 负载类型，这里是 JPEG 图像。在 RTP 协议中，payload 类型是用来识别编码数据的数字标识。
    8. rtpjpegdepay：
    9. rtpjpegdepay 是一个 RTP JPEG 解包器，它从 RTP 包中提取出 JPEG 编码的视频数据。
    10. jpegdec：
    11. jpegdec 是一个 JPEG 解码器，用于将 JPEG 编码的数据解码为原始视频帧。
    12. autovideosink：
    13. autovideosink 是一个视频显示接收器，它自动选择适合当前平台的方法来显示视频。

    H264编码

    jetson nano：作为服务器端

    IP:10.0.0.179

gst-launch-1.0 -v videotestsrc ! 'video/x-raw,width=(int)1280,height=(int)720,framerate=20/1' ! videoscale ! videoconvert ! x264enc tune=zerolatency bitrate=2048 speed-preset=superfast ! rtph264pay config-interval=1 pt=96 ! udpsink host=10.0.0.215 port=5004

    RTSP 实时流传输协议（Real Time Streaming Protocol, RTSP）

    是TCP/IP协议体系中的一个应用层协议，由哥伦比亚大学、网景和RealNetworks公司提交的IETF RFC标准。该协议定义了一对多应用程序如何有效地通过IP网络传送多媒体数据。RTSP在体系结构上位于RTP和RTCP之上，它使用TCP或UDP完成数据传输

    启动 deepstream

    import gi
gi.require_version('Gst','1.0')
gi.require_version('GstRtspServer','1.0')
from gi.repository import Gst, GObject, GstRtspServer

Gst.init(None)
mainloop = GObject.MainLoop()
server = GstRtspServer.RTSPServer()
factory = GstRtspServer.RTSPMediaFactory()
factory.set_launch(
    ' ( videotestsrc is-live=1 '
    '! video/x-raw,format=(string)NV12,width=(int)320,height=(int)180,framerate=20/1 '
    '! nvvidconv ! nvv4l2h264enc ! h264parse ! rtph264pay name=pay0 bitrate=40000 pt=96 ) '
)
# allow multiple connections
factory.set_shared(True)
mounts = server.get_mount_points()
mounts.add_factory('/live', factory)
print('rtsp://10.0.0.179:8554/live')
server.attach(None)
mainloop.run()

    更换摄像头数据源

    import gi
gi.require_version('Gst','1.0')
gi.require_version('GstRtspServer','1.0')
from gi.repository import Gst, GObject, GstRtspServer

Gst.init(None)
mainloop = GObject.MainLoop()
server = GstRtspServer.RTSPServer()
factory = GstRtspServer.RTSPMediaFactory()
factory.set_launch(
    ' ( nvarguscamerasrc '
    '! video/x-raw(memory:NVMM),width=(int)1280,height=(int)720,format=(string)NV12,framerate=(fraction)30/1 '
    '! nvv4l2h264enc ! h264parse ! rtph264pay name=pay0 bitrate=40000 pt=96 ) '
)

# allow multiple connections
factory.set_shared(True)
mounts = server.get_mount_points()
mounts.add_factory('/live', factory)
print('rtsp://192.168.2.9:8554/live')
server.attach(None)
mainloop.run()

    摄像头数据格式

    摄像头的数据输出格式一般分为RAW RGB和非raw RGB格式。Raw RGB是Sensor每个感光点感应到的RGB数值。Raw RGB经过ISP的处理还原出三原色，输出YUV或者RGB格式。

    摄像头数据格式 YUV

    1. YUV 是一种颜色编码方法：与RGB编码方法类似

    其中“Y”表示明亮度（Luminance 或Luma），也就是灰阶值，U和V 表示的则是色度（Chrominance 或 Chroma），用来表示，作用是描述影像色彩及饱和度，用于指定像素的颜色

    1. YUV格式好处

    可以避免相互干扰（没有UV信息一样可以显示完整的图像，因而解决了彩色电视与黑白电视的兼容问题）； 降低色度的采样率而不会对图像质量影响太大，降低了视屏是信号传输时对频宽（带宽）的要求。

    1. YUV数据格式有哪些

    YU12、1420、YV12、NV12、NV21、YUV420P、YUV420SP、YUV422P、YUV444P
