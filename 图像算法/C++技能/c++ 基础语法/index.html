    c++ 基础语法

    2.1 变量

    变量是内存地址的抽象

    变量有2 个重要的属性：

    Type：变量类型（integer、real number、string、Person、Student）

    Value: 变量值（1,3.14,”computer”）

    使用变量要首先声明**变量**

    int age //未初始化

int age = 18;  //c-风格初始化
int age (18);   // 构造函数用法 (OOP概念)
int age {18}   // c++11 标准：列表初始化语法

    2.2 基本类型

    字符型（character）：char

    • 作用：用于表示单个字符，如’@”’a，X’
    • 只占用1个字节
    • 语法：char c= a

    注意：

    • 需要和字符串区分开，用单引号不是双引号
    • 存储的是字符的ASCII编码，而不是字符本身

    整型（integer）：short、int、long、long long

    浮点型（floating point）：float、double

    布尔型（Boolean）：bool

    1 比特是存储的最小单位

    1 比特只能存储 0 或 1

    1 字节 = 8 比特

    1 字节能表达的模式数量 = 2^8 = 256 个

    可以用来存储0-255 数字

    适合存储字符/字母

    3.1 数组（array）

    • 长度大小固定，且不可以再修改
    • 所有元素是同一类型
    • 元素在内存中是连续存储的
      • 声明array时，内存中会分配一块区域来存储
      • 可以使用存储的索引位置来获取元素（从0开始，最后一个元素索引为 size -1）
      • 不检查是否超出区域（超出长度获取元素可能会报错）
    int student_scores[5] {92,78,100,86,65}; // 所有元素将初始化
int ages[10] {19,23}; // 前两个元素将初始化为19和23，其他元素为0

const int days_in_year {365};
double hi_temperatures[days_in_year] {0}; // 所有元素将初始化为0
int another_array[] {1,2,3,4,5,6}; // 数组大小为初始化时元素数量

    3.2 容器（vector）

    • 假设需要存储整个学校所有学生的考试成绩
    • 无法知道具体的学生数量
    • 解决办法：
      • 使用数组，数组大小设置的足够大
      • 使用vector 动态数组

    什么是容器？

    • c++ 标准模板库STL中的容器（container）
    • 长度大小可以随时缩放
    • 和数组类似的语法用法
    • 高性能
    • 越界检查
    • 很多有用的函数：sort, reverse, find等

    vector 声明

    #include <vector>
using namespace std;

vector<char> vowels;
vector<int> student_scores;

    vector<char> vowels (5); // 构造器将初始化为五个，默认每个值是 '\\0'
vector<int> student_scores (10); // 大小为10，与array不同，这10个整数会自动初始化为0

    vector<char> vowels {'a', 'e', 'i', 'o', 'u'};
vector<int> student_scores {100, 99, 98, 97, 96};
vector<double> hi_temperatures (365, 37.0); // 第一个数字是大小为365，第二个是所有值初始值为37.0

    vector 的性质

    • 动态调整大小
    • 所有元素是同一类型
    • 元素在内存中是连续存储的
      • 可以使用方便的索引位置来访问元素
      • 从0开始，最后一个元素索引为 size -1
    • []方式来获取元素，提供了函数做越界检查
    • 性能非常高
    • 所有元素会初始化为0

    vector 获取元素

    // 索引访问方式
vector_name[element_index]
student_scores[1]

vector<int> student_scores {100,98,97,96,95};
cout << "第1个分数" << student_scores[0] << endl;

    // vector 索引访问
vector_name.at(element_index)

student_scores.at(1);
vector<int> student_scores {100,98,97,96,95};
cout << "第1个分数" << student_scores.at(0) << endl;

student_scores.at(0) = 90; // 更新元素

    vector 追加元素

    // vector_name.push_back(new_element)

vector<int> student_scores {100,98,97}; // 大小为3
student_scores.push_back(80); // 100,98,97,80
student_scores.push_back(90); // 100,98,97,80,90
